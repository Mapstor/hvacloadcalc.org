# HVAC Calculators and Sizing Tools

This is the calculator hub. Every calculator on the site shows its math, cites its sources, and states its limits. The default state of each calculator opens with a worked example using reasonable defaults, so the methodology is visible before any input is changed. No account, no signup, no email gate — just inputs in, outputs out, formula shown.

Each calculator carries a planning-grade disclaimer above the fold. Results are educational; permit-grade load calculations require ACCA-approved software or a licensed HVAC contractor.

## Sizing calculators

- [Manual J load calculator](/tools/manual-j-calculator/) — whole-house heating and cooling loads with envelope math (conductive + infiltration + solar + internal gains)
- [Heat pump size calculator](/tools/heat-pump-size-calculator/) — dual-load sizing with balance point estimate and cold-climate (CCASHP) toggle
- [AC size calculator](/tools/ac-size-calculator/) — AC equipment sizing with window/portable/mini split/central recommendations
- [BTU calculator](/tools/btu-calculator/) — general-purpose BTU sizing for any room or space

## Envelope and insulation calculators

- [Attic R-value calculator](/tools/attic-r-value-calculator/) — depth × R-per-inch with multi-layer support, DOE zone targets, and upgrade depth recommendations

## How our calculators work

All calculators use pure-JavaScript front-end math. No server-side computation, no opaque transformations, no AI. Same inputs always produce same outputs. Each calculator's source formulas are documented inline, and we verify our Manual J calculator against ACCA reference cases.

See our [Manual J verification methodology](/methodology/how-we-verify-manual-j/) for the accuracy bands we claim and how we test against ACCA's published reference cases.
