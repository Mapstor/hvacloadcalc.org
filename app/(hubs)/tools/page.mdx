# HVAC Calculators and Sizing Tools

This is the calculator hub. Every calculator on the site shows its math, cites its sources, and states its limits. The default state of each calculator opens with a worked example using reasonable defaults, so the methodology is visible before any input is changed. No account, no signup, no email gate — just inputs in, outputs out, formula shown.

Calculators ship over Wave 2 of the build. The categories below show what's in scope.

## What this section covers

- **Load calculators** — Manual J-style room and home load calculation
- **Sizing calculators** — heat pump, AC, furnace, mini split capacity
- **Performance calculators** — balance point, SEER savings, fuel-use load
- **Duct calculators** — Manual D-style duct sizing, return air grille sizing
- **Reference calculators** — R-value, ACH, CFM, degree days, wet bulb

Each calculator carries a planning-grade disclaimer above the fold. Results are educational; permit-grade load calculations require a licensed HVAC contractor.
